## [minikube](minikube/start.md)

Minikube 是一个用于在本地运行 Kubernetes 的工具。它通过在你的个人电脑上的虚拟机 (VM) 或容器（如 Docker）中运行一个单节点 Kubernetes 集群，让你能够方便地进行学习和开发。

## [miniconda](miniconda/README.md)

Miniconda 是一个轻量级的 Anaconda 发行版，主要包含 Conda 包管理器和 Python。它是管理 Python 环境和依赖包的强大工具。

## [git](git/README.md)

Git 是目前世界上最先进的“分布式版本控制系统”
